Webpack-打包

因为最近做Vue小项目的进度到达了打包，所以今天总结一下经验把打包说一说😏

这里所说的打包是指使用Vue的npm run build命令进行打包，并且使用Sublime Text进行压缩，转化为webAPP

在使用npm run build进行打包之前，需要注意

  • 去除所有页面的console.log(),可使用正则表达式console\.log(.*\);? 将所有console.log语句去除
  • 将路由模式改成hash模式
  • 保证项目没有错误(vue的语法错误)，尽可能排除警告
  • 在根目录创建vue.config.js配置文件 touch vue.config.js
  • 对于4.5脚手架：在vue.config.js写入以下代码 打包后的文件可以本地方式打开
  • 对于5.0脚手架, 在vue.config.js写入以下代码 打包后的文件可以本地方式打开
  • 执行打包
    npm run build

使用APICloud打包APP(安卓、苹果)

1. 下载Sublime Text3
2. 为Sublime Text3安装package control插件    package control为了使用sublime Text3安装其他第三插件 同时按下 ctrl + shift + p 弹出安装插件面板
3. 如果成功安装package control插件后，可使用package control插件安装其他第三方插件
  • 同时按下 ctrl + shift + p 弹出安装插件面板，输入

  • 输入安装插件的名称 输入 APICloudLoader 进行安装
  • 安装完成后 新建一个空白文件夹，在文件加创建一个APICloud项目 ->空白应用(中文)
  • 将这个空白应用里的除 config.xml 文件外，其他文件和文件夹都删除
  • 将vue打包好的dist文件夹里的文件放到这个空白应用中
  • 在Sublime Text3左侧对文件夹右键，选择压缩widget包
  • 接下来就是打开 APICloud 创建应用
  • 将这个应用的id复制粘贴到config.xml里对应的 idA00000001
  • 对应用进行配置，创建证书，上传代码
  • 最后点击云编译，就可以创建出一个APP了

    赶紧学起来吧🤗
